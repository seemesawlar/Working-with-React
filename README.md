# Construction Company Website

A modern landing page for a construction business built with React, TypeScript, and Vite.

## Features

- Responsive homepage showcasing construction services
- Project gallery and featured portfolio sections
- Contact call-to-action for service inquiries
- Fast Vite-powered development workflow
- ESLint support with React and TypeScript

## Available scripts

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
```

## Local development

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal to preview the construction site.

## Production build

Build the optimized production bundle:

```bash
npm run build
```

The compiled files are output to the `dist` folder.

## Preview production build

```bash
npm run preview
```

## Linting

Run the project linter to check for code and style issues:

```bash
npm run lint
```
